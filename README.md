# Team Portfolio Website

## 📌 Description
This project is a final assignment for a Web Design course. It is a team portfolio website created by a group of six members to showcase our skills, roles, and collaboration.  
The website is built using HTML, CSS, and JavaScript with a focus on responsive design and user-friendly experience.
